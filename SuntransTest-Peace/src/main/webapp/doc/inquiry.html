参数

  • token:令牌符(必填项)
  • room_id:房间ID(必填项)
  • inquirytime:查询时间，格式yyyy-MM-dd，比如要查询2017年7月份的则为2017-07-01，八月为2017-08-01

响应

结果说明

节点说明：

  • code：表示响应结果状态，1表示查询成功，0表示未查询到信息
  • message：响应结果的文字说明

info节点说明：

  • user_id: 用户id
  • port: 通道号
  • name: 名字
  • created_at: 时间
  • message: 操作信息
  • status: 操作动作（打开或者关闭）

name+created_at+message构成完整信息：

简要示例 
		{
    "code": 1,
    "message": "Inquiry_StudentInfo_Byusername Success",
    "info": [
        {
            "user_id": 123,
            "port": 1,
            "name": null,
            "created_at": "2017-07-07 09:50:47",
            "message": "关闭了照明开关",
            "status": false
        }
    ]
}
